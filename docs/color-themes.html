颜色主题(Color Themes)

Framework7 默认有十种颜色的主题：

gray #8e8e93
white #ffffff
black #000000
lightblue #5ac8fa
yellow #ffcc00
orange #ff9500
pink #ff2d55
blue (default) #007aff
green #4cd964
red #ff3b30

如何使用不同颜色的主题

你只需要在需要使用的组件的父容器上加上 theme-[color] class即可。可以加在 body, view, page, navbar, toolbar 等上面，比如：

<body class="theme-red">
    ...
</body>
 
<div class="page theme-green">
    ...
</div>
 
<div class="list-block theme-pink">
    ...
</div>
 
<div class="navbar theme-orange">
    ...
</div>
 
<div class="buttons-row theme-yellow">
    ...
</div>

注意，颜色主题只会影响交互组件，比如链接、按钮、表单、图表等，并不会改变其他组件的文本颜色或者背景色。

工具类(Helper Classes)

每个颜色主题都有一个对应的工具类可以在主题之外使用：

  • color-[color] - 可以直接改变一个组件的颜色，比如:

    <a class="button color-red">Red button</a>
<i class="icon icon-back color-red">

  • bg-[color] - 可以直接改变一些组件的背景颜色（并不是所有组件的背景色都可以改）:

    <span class="badge bg-pink">14</span> - pink badge

  • border-[color] - 直接改变border的颜色:

    <div class="navbar border-white">...</div>

当然，你也可以混合着用:

<div class="navbar bg-lightblue color-white border-gray">...</div>

不过，在容器已经指定了颜色主题的情况下，你不能在里面再使用 helper class，比如这样就无效：

<div class="theme-red">
    <a href="#" class="color-blue">Link will be red, not blue</a>
</div>

布局主题(Layout Themes)

Framework7 现在多了两个新的 layout themes。在使用layout-themes之前我们需要先引入css文件:

Framework7 also has 2 new whole layout color themes. We need to include additional CSS files with these themes to use them:

<head>
    ...
    <link rel="stylesheet" href="path/to/framework7.min.css">
    <!-- It should be included After main Framework7 styles -->
    <link rel="stylesheet" href="path/to/framework7.themes.css">
</head>

有两个可选的 layout themes: "white" 和 "dark"。和使用color themes 一样，我们只需要在对应组件的容器上加上 layout-[theme] class即可。这个容器可以是 body, view,page, navbar, toolbar, list-block 等，比如：

<body class="layout-dark">
    ...
</body>
 
<div class="page layout-white">
    ...
</div>
 
<div class="list-block layout-dark">
    ...
</div>
